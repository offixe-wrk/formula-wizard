# Formula Builder

Excel Formula Builder is a web-based tool that helps users learn, preview, and build Excel formulas interactively. It provides a categorized list of Excel functions, detailed explanations, syntax examples, and an automatic formula builder where users can insert arguments and generate formulas without memorizing complex syntax.

---

### Features

#### 1. Interactive Formula Builder
- Select any Excel function from the function list.
- View its description, syntax, and required arguments.
- Fill in the argument input fields.
- The formula preview updates in real time.
- Copy the generated formula to your clipboard.

#### 2. Function Categories
Functions are grouped into:
- Math
- Logical
- Text
- Date
- Lookup
- Statistical
- Financial
- Engineering
- Information
- Database

This makes it easy to browse and learn.

#### 3. Responsive UI (Mobile Friendly)
- Sidebar becomes callapsible on small screens.
- Smooth transitions and clean layout.
- Touch-friendly navigation.

#### 4. Searchable Function List
- Search through all Excel functions instantly
- Clicking one loads its description and syntax.
